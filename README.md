# Personal Setup

This is my on the go containerized development setup having below tools ready
1. Docker (dind)
2. nodejs
3. npm
4. heroku-cli
5. git


## Setup
1. put your ssh keys inside .ssh directory
2. setup name and email in .env file

Bind local vol
```
export LOCAL_VOL=/path/to/directory # optional
```

## Build
```
make
```

## Run
```
make run
```
